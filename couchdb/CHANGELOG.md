# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-27

### Added

- Initial release of CouchDB add-on for Home Assistant
- Support for configurable port, username, and password
- Persistent data storage in Home Assistant config directory
- Web interface (Fauxton) access
- Multi-architecture support (aarch64, amd64, armv7)
- Comprehensive documentation

### Features

- CouchDB 3.x database server
- RESTful HTTP API access
- Web-based administration interface
- Automatic data persistence
- Secure admin authentication
- Single-node configuration optimized for Home Assistant
